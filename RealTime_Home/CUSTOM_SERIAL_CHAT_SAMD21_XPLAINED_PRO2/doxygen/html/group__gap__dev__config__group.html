◆ at_ble_set_dev_config()

at_ble_status_t at_ble_set_dev_config ( at_ble_dev_config_t *  config)

Set device configuration.

Parameters
[in]configPointer to configurations data structure
Note
Default values would be applied in case of not assign another configurations values.
  • Role : AT_BLE_ROLE_ALL
  • Renew Duration : AT_RENEW_DUR_VAL_MIN
  • Address : 0x1032547698BA
  • IRKey : 0x1032547698BADCFE1101321541761981BA1DC1FE
  • Address Type : AT_BLE_ADDRESS_PUBLIC
  • Attribute Config : 0x20, which means:
    1. Name Permission : AT_BLE_WRITE_DISABLE
    2. Appearance Permission : AT_BLE_WRITE_DISABLE
    3. Enable SPCS : false
    4. Enable Service Changed : true
    5. RFU : AT_BLE_WRITE_DISABLE
  • GAP Start Handle : AT_BLE_AUTO_ALLOC_HANDLE
  • GATT Start Handle : AT_BLE_AUTO_ALLOC_HANDLE
  • Maximum MTU : AT_MTU_VAL_RECOMMENDED
See also
at_ble_dev_config_t
Returns
Upon successful completion the function shall return AT_BLE_SUCCESS, Otherwise the function shall return at_ble_status_t
